MGPOPSIM

INPUT: 
	A user defined parameter file with fixed format. See para.txt for detaile.

OUTPUT: 
xxBV.out
	The id, pedigree, values of all individuals in generation xx.
xxMTYPE.OUT
	The marker genotypes of each individual in generation xx.	
xxQTYPE.out
	The QTL genotypes of each individual in generation xx.	
MMAP.out
	The marker map for the simulated data. 
	(SNP_name, SNP_id)
QMAP.out
	The  map for the simulated QTL. 
	Line 1 #_chromosome, #_markers on each chromosome, #_QTL)
	Line 2 QTL_id, qtl_effect
